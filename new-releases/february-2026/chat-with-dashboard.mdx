title: "Chat with a Dashboard"
---

The Chat with a Dashboard feature embeds a conversational AI copilot directly into your dashboard pages. This allows you to ask natural-language questions about your data, receive automated executive summaries, and identify key changes since your last visit. By keeping the chat experience within the dashboard, you can perform deep-dive analysis without losing the context of your visual reports.

## How it works

When you open a dashboard, the system automatically starts a chat session connected to the dashboard’s data. The AI detects the domain used by the dashboard widgets and uses it as context for the conversation.

<Steps>
  <Step icon="sidebar" title="Access the Chat">
    When you open a dashboard, a resizable chat sidebar typically appears on the right side of the page by default. You can also manually toggle the sidebar by clicking the **Sidebar** icon located in the **Dashboard Header** or by using the keyboard shortcut **Cmd+I** (macOS) or **Ctrl+I** (Windows).
  </Step>
  <Step icon="sparkles" title="Review Automated Insights">
    At the top of the chat interface, the system provides two specialized AI tiles:

    - **Executive Summary:** A streaming summary of all dashboard data that provides a narrative overview of your metrics.
    - **What Changed:** A time-diff comparison that highlights specific data deltas and trends since your previous visit.
  </Step>
  <Step icon="messages-question" title="Interaction and Follow-up">
    Below the automated tiles, you can use the full Wisdom chat experience to ask specific natural-language questions. If you want to analyze a specific widget, click the **...** kebab menu on that widget and select **Ask follow-up questions**. This action utilizes **context extraction** to attach the widget's specific data as a reference for your chat query.
  </Step>
  <Step icon="circle-plus" title="Update the Dashboard">
    If you are a dashboard creator, you can add new insights directly to the page from the chat. When the AI generates a chart or table you wish to keep, click the **Add to Dashboard** button on the chat artifact, or simply type "Add this chart to the dashboard" into the input field.
  </Step>
</Steps>

## Dashboard chat tools

The dashboard chat experience uses specialized tools that allow the AI to interact with the dashboard data and structure.

| Tool                    | Purpose                                                       |
| :---------------------- | :------------------------------------------------------------ |
| **SUMMARIZE_DASHBOARD** | Generates AI summaries of dashboard metrics and widget data.  |
| **DASHBOARD_CRUD**      | Creates, updates, or modifies dashboard widgets and elements. |
| **LIST_DASHBOARDS**     | Retrieves available dashboards and related metadata.          |

These tools allow the AI to analyze dashboard data, generate summaries, and create new visualizations that can be added directly to the dashboard.

## Key capabilities

Chat with a Dashboard provides several tools to streamline your data monitoring and creation workflows:

- **In-Context Summaries:** Receive an instant narrative overview of complex dashboards without manual interpretation.
- **Historical Comparisons:** Quickly identify what has changed in your data between sessions.
- **Widget-Specific Analysis:** Use the **Ask follow-up questions** feature to dive deeper into the specific logic of any individual chart.
- **Streamlined Editing:** Creators can build and iterate on dashboards by instantly adding chat-generated widgets.

## Target users

- **Admins:** Use the **Executive Summary** for high-level narrative overviews.
- **Explorers and Analysts:** Use the chat sidebar to ask ad-hoc questions about specific data points.
- **Dashboard Builders:** Use creator mode to quickly add new visualizations to a dashboard based on a chat conversation.

## FAQs

<AccordionGroup>
  <Accordion title="Do I need to start a new chat to ask questions about a dashboard?">
    No. The chat sidebar is automatically scoped to the dashboard’s data, allowing you to ask questions without leaving the page.
  </Accordion>
  <Accordion title="Can I ask questions about a specific widget?">
    Yes. From the widget menu, select **Ask follow-up questions** to attach that widget’s data to the conversation.
  </Accordion>
  <Accordion title="Can I add widgets from chat responses?">
    Yes. In creator mode, chat responses can generate artifacts such as charts that include an **Add to Dashboard** action.
  </Accordion>
  <Accordion title="What permissions are required?">
    The feature must be enabled through the `DASHBOARD_CHAT_ENABLED` feature flag, and users must have permission to read the dashboard data.
  </Accordion>
</AccordionGroup>
