---
title: "Delete Users From Workspace"
---

The `deleteUsersFromWorkspace` mutation removes one or more users from a workspace.

<Warning>
  This action is **irreversible**. Deleted users cannot be restored.
</Warning>

## Signature

```graphql
deleteUsersFromWorkspace(input: DeleteUsersFromWorkspaceInput!): DeleteUsersFromWorkspaceResponse!
```

## Arguments

<ParamField path="input" type="DeleteUsersFromWorkspaceInput!" required>
  Input object identifying the users to delete and the target workspace.
</ParamField>

### DeleteUsersFromWorkspaceInput fields

<ParamField path="ids" type="[ID!]!" required>
  The IDs of the users to delete. Use [listUsers](/integrations/embeddings/graphql-api/queries/user/list-users) to find user IDs.
</ParamField>

<ParamField path="workspaceID" type="ID!">
  The workspace to delete the users from. Required for multi-workspace deployments.
</ParamField>

## Response

<ParamField path="status" type="ResponseStatus!">
  Indicates whether the operation succeeded. See [ResponseStatus](/integrations/embeddings/graphql-api/objects/response-status).
</ParamField>

mutation DeleteUsersFromWorkspace($input: DeleteUsersFromWorkspaceInput!) {
