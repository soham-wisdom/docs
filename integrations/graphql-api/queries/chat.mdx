Retrieve conversation data and available analysis tools for chat interactions.

## conversationHistory

Retrieves the complete message history for a specific conversation, including both text content and visualizations.

### Signature

```graphql
conversationHistory(conversationId: ID!): ConversationHistoryResponse!
```

### Parameters

<ParamField path="conversationId" type="ID!" required>
  The unique identifier of the conversation to retrieve messages from.
</ParamField>

### Response Type

Returns a `ConversationHistoryResponse` object containing an array of conversation messages.

## chatTools

Retrieves the list of available analysis tools that can be used during chat interactions for a specific domain.

### Signature

```graphql
chatTools(domainId: ID!): [ChatTool!]!
```

### Parameters

<ParamField path="domainId" type="ID!" required>
  The unique identifier of the domain to get available tools for.
</ParamField>

### Response Type

Returns an array of `ChatTool` objects representing available analysis capabilities.

## Authentication

All chat queries require authentication via bearer token. Include your API token in the Authorization header:

```
Authorization: Bearer YOUR_API_TOKEN
```
