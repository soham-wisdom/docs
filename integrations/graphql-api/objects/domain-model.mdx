title: "DomainModel"

This page describes the schema for domain objects that organize related data sources.

## DomainModel

<ParamField path="id" type="ID!" required>
  Unique identifier for the domain.
</ParamField>

<ParamField path="version" type="Int!" required>
  Version number of the domain for change tracking.
</ParamField>

<ParamField path="name" type="String!" required>
  Human-readable name for the domain.
</ParamField>

<ParamField path="description" type="String">
  Optional description explaining the purpose and contents of the domain.
</ParamField>

<ParamField path="databases" type="[DatabaseModel!]!" required>
  List of databases included in this domain. See [DatabaseModel](#databasemodel).
</ParamField>

<ParamField path="knowledge" type="[String!]!" required>
  Domain-specific knowledge and context information.
</ParamField>

<ParamField path="domainSystemInstructions" type="String">
  Custom system instructions for AI interactions within this domain.
</ParamField>

<ParamField path="fyStartMonth" type="Int">
  Fiscal year start month (1-12) for financial calculations.
</ParamField>

<ParamField path="vizConfig" type="VizConfig!" required>
  Visualization configuration settings. See [VizConfig](#vizconfig).
</ParamField>

<ParamField path="codegenConfig" type="CodegenConfig!" required>
  Code generation configuration settings. See [CodegenConfig](#codegenconfig).
</ParamField>

## DatabaseModel

<ParamField path="id" type="ID!" required>
  Unique identifier for the database.
</ParamField>

<ParamField path="version" type="Int!" required>
  Version number of the database for change tracking.
</ParamField>

<ParamField path="name" type="String!" required>
  Name of the database.
</ParamField>

<ParamField path="domainId" type="ID!" required>
  Identifier of the domain this database belongs to.
</ParamField>

<ParamField path="connection" type="Connection">
  Connection details for accessing this database. See [Connection](/integrations/graphql-api/objects/connection).
</ParamField>

<ParamField path="isUserCSVDatabase" type="Boolean!" required>
  Whether this database was created from user-uploaded CSV files.
</ParamField>

## Configuration Objects

### VizConfig

<ParamField path="alwaysUseCompactNumberFormat" type="Boolean!" required>
  Whether to always use compact number formatting in visualizations.
</ParamField>

<ParamField path="openShowDataPanelByDefault" type="Boolean!" required>
  Whether to show the data panel by default in visualizations.
</ParamField>

### CodegenConfig

<ParamField path="chatThinkingEffort" type="ChatThinkingEffort!" required>
  Level of thinking effort for AI code generation. See [ChatThinkingEffort](#chatthinkingeffort).
</ParamField>

## Enums

### ChatThinkingEffort

The level of thinking effort for AI interactions.

**Values:**
- `Low` - Minimal thinking effort for simple queries
- `Medium` - Moderate thinking effort for standard analysis
- `High` - Maximum thinking effort for complex analysis
