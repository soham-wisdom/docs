Core objects for managing domains, data connections, and analysis tools in Wisdom AI.

## Overview

Domain objects provide the foundation for organizing and accessing your data sources. They enable you to group related tables from various connections and manage the tools available for analysis.

## Enums

### ChatToolType

The type of analysis tool.

**Values:**
- `PYTHON_INTERPRETER` - Python code execution environment
- `SQL_GENERATOR` - SQL query generation tool
- `CHART_GENERATOR` - Data visualization tool
- `DATA_ANALYZER` - General data analysis tool
- `CUSTOM` - Custom tool implementation
