title: 'Chat with a Dashboard'
---

This guide explains how to use the Chat with a Dashboard embeds a conversational AI copilot directly into dashboard pages. Users can ask natural-language questions about their dashboard data, get AI-generated executive summaries, see what changed since their last visit, and even add new widgets to the dashboard, all without leaving the page.


## Get started
Before you begin, ensure you have the following:

- A dashboard you want to chat with. If you haven't created a dashboard yet, there are two ways to do so, see the [Turn Answers into Dashboards](/using-wisdom-ai-everyday/turn-answers-into-dashboards) or the [Build Dashboards](/improve-wisdom-ai-responses/build-dashboards) guide.

## Ask a question about your dashboard data
When the sidebar opens, two AI generated tiles appear at the top:

* **Executive Summary**: A streaming narrative summary of all data in the current dashboard.
* **What Changed**: A comparison highlighting differences since your last visit.

These tiles load automatically. No action is required to generate them.

![dashboard-tiles](/images/dashboard-tiles.png)

To initialize a chat with a dashboard, follow these steps:

1. Click o **Dashboards** on the sidebar.
2. Select the dashboard you wish to chat with.

![select-dashboard-image](/images/dashboard1.png)

3. Click the chat input at the bottom of the sidebar and type a natural-language question about the data shown on the dashboard.
4. Press **Enter** or click the send button.

![dashboard-input](/images/dashboard-input.png)

The AI agent will respond with data, tables, or charts scoped to the dashboard's domain.

![Chat with a question typed in, and an AI response ](/images/dashboard-response.png)

## Ask a follow-up question about a specific widget

You can attach a specific widget's data as context for a more targeted question.

1. Hover over the widget you want to investigate.
2. Click the **...** menu on the widget.
3. Select **Ask follow-up questions**.

   The widget's data is attached as a reference in the chat input.

4. Type your question and press **Enter**.

[SCREENSHOT: Widget with the "..." menu open and "Ask follow-up questions" highlighted]

## Add a widget to the dashboard from chat (creator mode)

If you have creator access to the dashboard, you can add new widgets directly from a chat response.

1. Ask the AI to create a visualization — for example, *"Show me monthly revenue by region as a bar chart."*
2. When the response artifact appears, click **Add to Dashboard**.

   The new widget is added to the dashboard.

Alternatively, you can ask the AI directly: *"Add this chart to the dashboard."*
